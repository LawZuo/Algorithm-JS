### Array Methods

```js
concat      连接2个或更多数组，并返回结果
join        将所有的数组元素连接成一个字符串
reverse     反转数组
toString    将数组作为字符串返回
valueOf     将数组作为字符串返回
slice       传入索引值，将数组里对应索引范围内的元素作为新数组返回
sort        按照字母顺序对数组排序，支持传入指定排序方法的函数作为参数

push        在数组尾部追加元素
pop         删除尾部最后一个元素
shift       删除首部的元素
unshift     在数组首部添加元素
splice      删除/插入元素

// 高阶函数
indexOf     返回第一个与给定参数相等的元素的索引，没有返回-1
lastIndexOf 返回在数组中搜索到的与给定参数相等的元素的索引里最大的值

every       对数组中每个元素给定运行函数，如果所有元素都满足该函数，则返回true
some        对数组中每个元素给定运行函数，如果任一元素返回true，则返回true

forEach     对数组中的每个元素运行给定函数，无返回值
filter      对数组中的每个元素运行给定函数，返回该函数会返回true的元素组成的数组
map         对数组中的每个元素运行给定函数，返回每次函数调用的结果组成的数组

//ES6
copyWithin  复制数组中一系列元素到同一数组指定的起始位置
includes    如果数组中存在某个元素则返回true， 不存在返回false
find        根据回调函数给定的条件从数组中查找元素，返回找到的元素
findIndex   根据回调函数给定的条件从数组中查找元素，返回找到元素的索引值
fill        用静态值填充数组
Array.from()根据已有数组创建一个新数组
Array.of()  根据传入的参数创建一个新数组
keys        返回包含数组中所有键的新数组
values      返回包含数组中所有值的新数组
entries     返回包含数组中所有键值对的新数组
```
